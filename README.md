# Greenway Goods

A premium web experience for Greenway Goods, delivering pure and natural products from the hills of Tamil Nadu.

## Features
- **Modern UI**: Designed with smooth animations using GSAP and fully responsive for all devices.
- **Product Showcase**: Detailed, parallax-enabled product presentation.
- **Interactive Storytelling**: Scroll-driven storytelling and statistics.

## Development

This project is built with Next.js and Tailwind CSS.

### Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
